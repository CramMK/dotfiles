## Features
+ Distro: `Fedora 31`
+ Shell: `zsh + oh-my-zsh`
+ WM: `i3-gaps (home), i3 (work)`
+ Terminal: `xterm`
+ Font: `Inconsolata Nerd Font Mono`
+ Vim-Theme: `Nord`
+ Vim-Bar: `airline`
+ Xresources: `Nord`

## Requirements
+ (`i3`, `Terminal`, `zsh`)
+ Bar: `polybar`
+ Browser: `chromium`
+ Mail: `Thunderbird`
+ Transparency: `picom`
+ Backlight: `light`
+ Notifications: `dunst`
+ Audio Control: `playerctl`
+ App Launcher: `rofi`
+ Audio Visualizer: `cava`
+ Wallpaper: `teh`
+ Filebrowser: `vifm`

## Profiles
+ `laptop`: All of the dotfiles
+ `work`: Everything I need for work
+ `dev`: Working on remote (dev-)servers
